## Installation

Need the following python libraries for Time-Series:
- numpy
- pandas
- Pathlib
- statsmodel.api
- statsmodels.tsa.arima_model
- arch

Need the following python libraries for Regression:
- numpy
- pandas
- Pathlib
- sklearn.linear_model
- sklearn.metrics
